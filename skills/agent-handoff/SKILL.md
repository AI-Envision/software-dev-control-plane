# Agent Handoff Skill

Use this skill when handing work from one agent or human to another.

Handoff should include:
- Repository
- Branch
- Commit
- Working tree state
- Task
- Files changed
- Commands run
- Results
- Risks
- Next step

Gotchas:
- A handoff without branch and commit is not reproducible.
- A handoff without validation results is not trustworthy.
- Distinguish done, not done, and not checked.
